# Custom Language 사용 방법

[`README.md`](../../README.md)의 파이프라인을 통해 실제로 실행되는 언어 문법과 기능을 예시로 정리합니다.

## 연산자(Operator)

```
print 1 + 2;          // 3
print "a" + "b";      // ab  (문자열 + 문자열은 이어붙이기)
print 7 % 2;          // 1
print -3 + 1;         // -2
print !false;         // true
print 1 < 2;          // true
var arr = Array(2);
arr[0] = 1;
print arr[0];         // 1
```

### 지원 연산자

| 분류 | 연산자 | 비고 |
| --- | --- | --- |
| 산술 | `+` `-` `*` `/` `%` | `+`는 숫자+숫자 또는 문자열+문자열만 허용(혼용 시 런타임 에러). `-`는 이항 뺄셈/단항 부호 반전 겸용. `/`, `%`는 0으로 나누면 각각 `"Division by zero."`, `"Modulo by zero."` 런타임 에러. 모든 숫자는 내부적으로 `double`. |
| 비교 | `<` `>` | 이 둘만 지원. |
| 논리 | `!` | 단항 부정만 지원. |
| 대입 | `=` | 변수 / `obj.field` / `arr[i]`에만 대입 가능. |
| 기타 | `instanceof` `.` `[]` `()` | 인스턴스 검사, 멤버 접근, 인덱싱, 함수 호출. |

### 지원하지 않는 연산자

| 분류 | 연산자 | 비고 |
| --- | --- | --- |
| 비교 | `==` `!=` `<=` `>=` | 토큰/문법 자체가 없어 사용 불가. |
| 논리 | `&&` `\|\|` `and` `or` | 파서가 실제로 생성하지 않음(AST에 `LogicalExpr`는 남아있지만 렉서가 토큰을 만들지 않는 죽은 코드). |
| 삼항 | `?:` | 없음. |
| 비트 | `&` `\|` `^` `~` `<<` `>>` | 없음. |
| 복합 대입 / 증감 | `+=` `-=` `*=` `/=` `%=` `++` `--` | 없음. |

> 위 표는 대표적인 예시일 뿐이며, 표에 없는 연산자라도 [지원 연산자](#지원-연산자) 표에 명시되지 않은 것은 모두 지원하지 않습니다.

### 연산자 우선순위

| 우선순위 | 분류 | 연산자 |
| --- | --- | --- |
| 1 (높음) | 기본식 | 리터럴, 식별자, 괄호 `(...)` |
| 2 | 후위 | 호출 `()`, 멤버 `.`, 인덱스 `[]` |
| 3 | 단항 | `-`, `!` |
| 4 | 곱나눗셈/나머지 | `*`, `/`, `%` |
| 5 | 가감 | `+`, `-` |
| 6 | 비교 | `<`, `>`, `instanceof` |
| 7 (낮음) | 대입 | `=` |

## 제어문(Control Flow)

```
for (var i = 0; i < 3; i = i + 1) {
  print i;             // 0 1 2
}

if (x < 0) {
  print "negative";
} else {
  print "non-negative";
}
```

### 지원하는 제어문

| 문법 | 형태 | 비고 |
| --- | --- | --- |
| `if` | `if (조건) 문 (else 문)?` | else는 가장 가까운 if에 연결(dangling-else). |
| `for` | `for (초기화?; 조건?; 증감?) 문` | C 스타일, 세 절 모두 생략 가능. |
| 블록 | `{ 문* }` | 새로운 렉시컬 스코프를 만듦. |

### 지원하지 않는 제어문

| `while` | `do-while` | `switch`/`case` | `break` | `continue` | `foreach` |
| --- | --- | --- | --- | --- | --- |
| ✗ | ✗ | ✗ | ✗ | ✗ | ✗ |

## 리터럴과 타입

### 값 타입

| 타입 | 리터럴/표기 | 비고 |
| --- | --- | --- |
| 숫자 | `1`, `3.5` | 모두 내부적으로 `double`(정수/실수 구분 없음). `.5`처럼 정수부 없는 표기는 숫자로 렉싱되지 않음(정수부 최소 한 자리 필요). |
| 문자열 | `"..."` | 이스케이프 시퀀스(`\n`, `\"`, `\\` 등) 미처리 — 백슬래시는 문자 그대로 남음. 닫는 `"` 없으면 파일 끝까지 소비. |
| 불리언 | `true`, `false` | 소문자 키워드만 인정. |
| 값 없음 | (키워드 없음) | `nil`/`null` 키워드는 없음. 초기화 안 한 `var x;`, 값 없는 `return;`에서만 암묵적으로 발생. |
| 함수 / 클래스 / 인스턴스 / 배열 | `Func`, `Class`, `Class(...)`, `Array(n)` | 아래 각 섹션 참고. |

### 지원하지 않는 리터럴 문법

| 항목 | 비고 |
| --- | --- |
| 주석(`//`, `/* */`) | 처리되지 않음 — 소스에 주석을 쓰면 `/`가 나눗셈 연산자로 렉싱되어 파스 에러 발생 가능. (이 문서 코드 예시의 `// 10` 같은 주석은 설명용 표기일 뿐, 실행 가능한 소스에는 넣을 수 없음) |
| 배열 리터럴(`[1, 2, 3]`) | 없음 — `Array(n)`으로만 생성. |

## 기본 지원 기능

| 종류 | 이름 | 비고 |
| --- | --- | --- |
| print | `print <expr>;` | 유일한 출력 수단. 불리언은 `"true"/"false"`, 문자열은 그대로, 숫자는 `double` 그대로 출력. 출력 뒤 줄바꿈 포함. |
| printn | `printn <expr>;` | `print`와 동일하나 출력 뒤 줄바꿈을 붙이지 않음. |
| 전역 함수 | 없음 | `clock()`, `len()`, `input()` 등 사전 등록된 전역 함수가 하나도 없음. |

## 함수(Function)

```
Func add(a, b) {
  return a + b;
}
print add(3, 7);   // 10

Func greet() {
  return;          // 값 없는 반환
}

Func fact(n) {
  if (n < 2) return 1;
  return n * fact(n - 1);
}
print fact(5);      // 120
```

| 상황 | 결과 |
| --- | --- |
| 함수 밖에서 `return` 사용 | 정적 에러 |
| 파라미터 이름 중복(`Func foo(a, a)`) | 정적 에러 |
| 호출부 인자 개수가 함수 선언과 다름(`Func foo(a,b,c){} foo(1,2);`) | 정적 에러 — `var x = foo(1,2);`, `print foo(1,2);`처럼 다른 표현식에 중첩된 호출이나 대입/이항·단항·논리 연산의 좌우항, 괄호 안 등 표현식이 나타날 수 있는 모든 자리까지 정적으로 검사 |
| 함수가 아닌 값을 호출(`var x = 1; x();`) | 런타임 에러 `"Can only call functions."` |
| `DfineShell`(REPL)에서 선언/호출이 다른 줄에 있음 | 인자 개수 정보를 줄 사이에 유지하여 여전히 검출됨 |

## 클래스(Class)

```
Class Animal {
  init(name) {
    This.name = name;
  }
  speak() {
    print This.name;
  }
}

Class Dog : Animal {
  speak() {
    Super.speak();
    print "Woof";
  }
}

var d = Dog("Rex");
d.speak();               // Rex
                          // Woof
print d instanceof Animal; // true
```

| 문법 | 설명 |
| --- | --- |
| `Class A : B` | 단일 상속(다중 상속 불가) |
| `This` | 자기 인스턴스 필드/메서드 접근 |
| `Super.method(...)` | 부모 클래스 메서드 호출 |
| `instanceof` | 인스턴스가 특정 클래스(또는 그 조상)의 인스턴스인지 검사 |

### 정적 에러가 되는 경우

| 상황 | 예시 |
| --- | --- |
| `This`/`Super`를 클래스 밖에서 사용 | — |
| 자기 자신을 상속 | `Class A : A` |
| 부모 없는 클래스에서 `Super` 사용 | — |
| `init`에서 값 있는 `return` 사용 | `init() { return 1; }` |

## 정적 배열(Array)

```
var arr = Array(3);
arr[0] = 10;
arr[1] = 20;
print arr[0] + arr[1];   // 30
```

| 문법 | 설명 |
| --- | --- |
| `Array(n)` | 크기가 고정된 배열 생성 |
| `arr[i]` | 읽기 |
| `arr[i] = v` | 쓰기 |

| 상황 | 결과 |
| --- | --- |
| 다차원 배열 | 지원 안 함 |
| `Array(a)`/`arr[i]`에서 `a`/`i`가 미선언 변수 | 정적 에러 |
| 인덱스가 숫자가 아니거나 정수가 아니거나 범위를 벗어남 | 런타임 에러 |

## import

```
import "utils.txt" alias utils;
print utils.helper(1, 2);
```

- 다른 소스 파일을 모듈로 불러와 `alias`로 지정한 이름을 통해 그 모듈의 최상위 선언에 접근합니다.

### 정적 에러가 되는 경우

| 상황 |
| --- |
| 반복문 안에서의 import |
| 같은 스코프에서의 중복 import/alias 충돌 |
| 순환 import |

## 실행 전 최적화

| 최적화 | 구현 | 설명 |
| --- | --- | --- |
| 상수 폴딩 | `ConstantFolder` | 리터럴끼리만 이루어진 사칙연산·비교연산·단항 마이너스를 실행 전에 미리 계산해 `LiteralExpr`로 치환. 변수가 하나라도 섞여 있으면 그대로 둠. |
| 정적 바인딩 | `Resolver` | 변수 참조가 몇 단계 위 스코프에 있는지(`distance`)를 미리 계산해 캐싱, 실행 시 매번 스코프 체인을 거슬러 올라가지 않고 즉시 접근. |
